Everything here is mostly ready to use, i've included almost all my configs.
here is the list of everything that's included:

1. Hyprland: My main WM config with some custom scripts for screenshots and more.
2. Waybar: A polished top bar with custom modules.
3. Foot: My primary terminal configuration.
4. Mako: My current notification daemon.
5. Fastfetch: Custom fastfetch config. I use "fastfetch -c examples/13" and add an alias for it in .bashrc called "neofetch"
6. Fuzzel: App launcher and menus.
7. wlogout: Logout menu.

Here is what I've left out for manual installation or external cloning:

1. Alacritty: I get it from tony, use: "git clone https://github.com/tonybanters/alacritty"
2. Waypaper: Can be installed through the AUR with "yay -S waypaper"
3. Ranger: I find the defaults sane.

Colorscheme: Catppuccin Mocha
WM: Hyprland (I am thinking about making more dotfiles for other WMs and possibly DEs)
Distro: Arch on main computer, NixOS on a thinkpad x1 carbon
Terminal(s): For daily-use it's Foot, For SSH and other stuff it's Alacritty

NOTE: Change some settings in files such as 'Hyprland.conf' for monitors or other things so that they work correctly for you!
You can also use my dotfiles as a base for your dotfiles and later upload them to github if you'd wish to :D
